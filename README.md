# CS2 Skin Inventory Tracker (V 1.0)

A Python CLI application to track Counter-Strike 2 skins, calculate profit, and manage inventory.

## Features

- Add skin
- Show inventory
- Search skin
- Update skin
- Delete skin
- Sort by profit
- Inventory summary
- JSON database

## Future Development

- Steam Market API integration
- Automatic price updates
- Web dashboard#cs2-skin-inventory
